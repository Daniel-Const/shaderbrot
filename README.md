# Shaderbrot

Playing around with Fragment Shaders over a Mandelbrot plot. C / Raylib

![Alt Text](./shaderbrot-demo.gif)

## How to run

1. Ensure [raylib](https://www.raylib.com/) is installed
2. ```bash
   make
   ./shaderbrot
   ```
